04-Bit

Xiaosong Tomas Wen 2021-06-16 13:40:47
Categories： data structure Tags： java bit

Bit Representation

  • Positive int:
    • 5: 0b 0000 0000 0000 … … 0000 0101
    • Java MAX_INT: 2^31 - 1 = 2147483647
  • Negative int:
    • Two’s complement: (取反+1）
    • -5: 0b 1111 1111 1111 … … 1111 1011
  • Conversion between different type:

JAVA:

  • bytes: 8 bits signed
  • short: 16 bits signed
  • int: 32 bits signed
  • long: 62 bits signed
  • char: 16 bits unsigned
  • Keep the same bit widths: binary representation does NOT change
  • Extending bit width: prepend zero if the sources is unsigned; prepend sign otherwise
  • Truncate the widths: Truncate the high bits:

Bit Operation:

  • & and

  • | or

  • ~ not

  • ^ xor

  • << left shift

  • > > right shift, fill the sign after shift

  • > > > logical right shift

Common usage:

Bit Sester:

get k-th bit of x: (x >> k) & 1

Bit Setter (Resetter):

set k-th bit of x to 1: x |= (1 << k)

set k-th bit of x to 0: x &= ~(1 << k)

Operator Priority:

  1. postfix: exp++, exp–
  2. unary: ++exp, –exp, +exp, -exp, ~, !
  3. multiplicative: *, /
  4. additive: + / -
  5. shift: <<, >>, >>>
  6. relational: <, >, <=, >=, instanceof
  7. equality: ==, !=
  8. &
  9. ^
  10. |
  11. &&
  12. ||
  13. temary: ? :
  14. assignment: =, +=, -= ….
