Tree

Balanced binary tree:

The height of the left and right subtrees of every node differ by 1 or less

  • If a tree has n number of nodes, and it is balanced, then the height(level) of the tree = O(log(n))
  • CAVEAT: if a tree has n nodes, but we don’t known if it is balanced, then the height(level) of the tree= O(n)

Complete binary tree:

A binary tree in which every level, except possible the last, is completely filled, and all nodes are as left as possible.

  • If a tree is a complete tree, then it must be a balanced tree.

Binary Search tree:

For every single node in the tree, the values in its left subtrees are all smaller than its value, and the values in its right subtree are all larger than its value.

  • The in-order sequence of a BST must form an ascending order
